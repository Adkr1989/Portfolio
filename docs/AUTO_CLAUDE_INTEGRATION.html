What is Auto-Claude?

An autonomous multi-agent coding framework that automates development. Agents create specs, implement code, run QA, and merge changes.

Parallel Execution

Up to 12 concurrent agent terminals

Isolated Workspaces

Git worktrees protect main branch

Self-Validating QA

Built-in quality checks

AI-Powered Merge

Automatic conflict resolution

Desktop App

Electron UI with Kanban board

Memory Layer

Cross-session context (disabled)
